### POSTGRESQL KOMUTLARI #######################################################

--- Tanım ----------------------------------------------------------------------

Bu komutlar hem standart SQL komutlarını hem de PostgreSQL'e ait olan bazı 
komutları içeriyor.
--------------------------------------------------------------------------------

--- SELECT version() -----------------------------------------------------------

Sisteme kurulu olan PostgreSQL'in sürümünü ve ilgili detayları gösterir.

SELECT version();
                          version
------------------------------------------------------------
 PostgreSQL 14.1, compiled by Visual C++ build 1914, 64-bit
(1 row)
--------------------------------------------------------------------------------

--- SELECT current_date --------------------------------------------------------

Sistemin tarihini gösterir.

SELECT current_date;
 current_date
--------------
 2022-01-23
(1 row)
--------------------------------------------------------------------------------
