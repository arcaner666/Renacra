text-transform

Yazıdaki harfleri tamamını veya yalnızca kelimelerin baş harflerini büyük/küçük yapar.

text-transform: lowercase; => TÜM YAZIYI KÜÇÜK YAPAR.

text-transform: uppercase; => tüm yazıyı büyük yapar.

text-transform: capitalize; => sadece kelimelerin baş harflerini büyük yapar.

text-transform: none; => kelimeler nasıl girilmişse öyle bırakır.

text-transform: inherit; => Parent'ından miras alır.

capitalize bir sayıdan sonra gelen ilk harfi büyük yapmaz. Bunun amacı tarihleri yazarken bozulmayı önlemektir.

text-transform: capitalize; => This text includes a date: February 4th, 2015.

capitalize sadece kelimelerin ilk harflerini büyük yapar. Zaten büyük veya küçük olan diğer harflere dokunmaz. Bu sayede kısaltmalar gibi şeyleri rahatça yazabiliriz.

text-transform: capitalize; => This text includes an acronym: CSS.
