width

İçeriğin genişliğini belirler.

Değer olarak sadece uzunluk birimleri kabul eder.

width: 800px;

width: 20em;

width: 80ch;

width: 40cm;

width: 50vh;

width: 40rem;

width: 100%;

Eğer kapsayıcı bloğun genişliği ayarlanmamışsa kapsayıcı blok tüm ekranı kaplar.

Child => width: 400px;

Eğer blok içinde bir içerik varsa içerik sol taraftan başlayarak kendi genişliği kadar genişlik kaplar.

Eğer bloğun içerik kısmı ayarlanan genişlikten daha fazla alana ihtiyaç duyuyorsa içeriğin davranışını overflow özelliği belirler.

Parent => width: 800px;

Child => width: 800px;

Parent ve child'ın genişliği eşit.

Parent => width: 800px;

Child => width: 900px;

overflow ayarlanmadığı için child'ın genişliği sağa doğru 100px taştı.

Parent => width: 800px; overflow: hidden;

Child => width: 900px;

Parent => width: 800px; overflow: scroll;

Child => width: 1000px;

Parent => width: 800px; overflow: auto;

Child => width: 1000px;

Eğer bir child'ın genişliğine auto değeri girilirse genişlik, özellikle belirtilmediği sürece child'ın içeriğinin genişliğine eşit olacaktır.

Parent => width: 800px;

Child => width: auto;

Eğer parent'ın genişliğine ölçülebilir bir genişlik, child'ın genişliğine ise yüzde bir değer verilirse child'ın genişliği parent'ınkine göre oranlanır.

Parent => width: 800px;

Child => width: 95%;
