color

Yazının ve yazı dekorasyonunun rengini belirler.

Değer olarak isimlendirilmiş renkler alabilir.

color: red;

Değer olarak Hex renkler alabilir.

color: #00FFFF;

Değer olarak RGB veya RGBa renkler alabilir.

color: rgb(80, 200, 100); ya da color: rgba(200, 34, 206, 0.7);

Değer olarak HSL veya HSLa renkler alabilir.

color: hsl(54, 68%, 43%); ya da color: hsla(54, 68%, 43%, .52);
