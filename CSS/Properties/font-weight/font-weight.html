font-weight

Yazının genişliğini ya da kalınlığını belirler. font-family tarafından sunulan font face'lere bağlı olarak ya da tarayıcıya bağlı olarak değişkenlik gösterebilir.

Değer olarak aşağıdaki kelimeleri alabilir.

font-weight: normal;

font-weight: bold;

Parent veya child'a göre göreceli olarak değiştirilebilir. Tam tersi de olabilir. bolder ve lighter child'lara miras olarak geçmez. Bunun yerine parent'ın hesaplanmış yazı kalınlığı miras olarak geçer.

Parent => font-weight: normal;

Child => font-weight: bolder; => bold

Parent => font-weight: bold;

Child => font-weight: lighter; => normal

Değer olarak aşağıdaki sayıları da alabilir.

font-weight: 100;

font-weight: 200;

font-weight: 300;

font-weight: 400;

font-weight: 500;

font-weight: 600;

font-weight: 700;

font-weight: 800;

font-weight: 900;

normal 400'e, bold'da 700'e eşittir.

font-weight: 400; = font-weight: normal;

font-weight: 700; = font-weight: bold;

400 ve 700 dışındaki font-weight'lerinde çalışması için ilgili font-family'nin desteklemesi gerekmektedir.

Eğer bir font'un normal(400) ve bold(700) versiyonları bulunuyorsa tarayıcı bunları kullanır. Eğer bu ikisinden birisi mevcut değilse tarayıcı bunları kendisi taklit etmeye çalışır. Fakat diğer yazı kalınlıkları eğer mevcut değillerse tarayıcı onları taklit etmez.
