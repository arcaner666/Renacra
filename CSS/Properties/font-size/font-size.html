font-size

Yazının boyutunu ya da yüksekliğini belirler. Sadece uygulandığı fontu etkilemekle kalmaz em, rem ve ex uzunluk birimlerinin hesaplanmasını da sağlar.

Değer olarak kelime (absolute keywords) alabilir.

font-size: x-small;

font-size: xx-small;

font-size: small;

font-size: medium;

font-size: large;

font-size: x-large;

font-size: xx-large;

Eğer parent'ın font-size'ı small ise child'ın font-size'ını larger yaparsak child'ın font-size'ı medium olarak hesaplanır. Tam tersi de yapılabilir.

Parent => font-size: small; Child => font-size: larger;

Parent => font-size: medium; Child => font-size: smaller;

Yüzde değer vererek child'ın font-size'ı parent'ın font-size'ına göre oranlanabilir.

Parent => font-size: 24px; Child => font-size: 130%;

em değeri child'ların yazı boyutunun parent'ın yazı boyutuna göre göreceli olarak değişmesini sağlar. Bu durum child'ların yazı boyutunun her zaman parent'a bağlı olmasını sağlar. em kullanmanın birçok avantajı vardır.

Parent => font-size: 24px; Child => font-size: 2em;
