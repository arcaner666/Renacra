font-size

Yazının boyutunu ya da yüksekliğini belirler. Sadece uygulandığı fontu etkilemekle kalmaz em, rem ve ex uzunluk birimlerinin hesaplanmasını da sağlar.

Değer olarak kelime (absolute keywords) alabilir.

font-size: xx-small;

font-size: x-small;

font-size: small;

font-size: medium;

font-size: large;

font-size: x-large;

font-size: xx-large;

Eğer parent'ın yazı boyutu small ise child'ın yazı boyutunu larger yapmamız durumunda child'ın son yazı boyutu medium olarak hesaplanır. Yani child'ın yazı boyutu parent'ınkine göre hesaplanır. Tam tersi de yapılabilir.

Parent => font-size: small;

Child => font-size: larger; => medium

Parent => font-size: medium;

Child => font-size: smaller; => small

Yüzde değer vererek child'ın yazı boyutu parent'ın yazı boyutuna göre oranlanabilir.

Parent => font-size: 24px;

Child => font-size: 130%; => 32px

em değeri child'ların yazı boyutunun parent'ın yazı boyutuna göre göreceli olarak değişmesini sağlar. Bu durum her child'ın yazı boyutunun her zaman parent'ına bağlı olarak değişmesini sağlar. em kullanmanın birçok avantajı vardır.

Parent => font-size: 12px;

Child => font-size: 2em; => 2em = 24px

Child'ın Child'ı => font-size: 2em; => 2em = 48px

rem değerini alan bir eleman kendisine bütün parent'ları tarafından uygulanan tüm font-size kurallarını görmezden gelerek kök (root) elemanın yazı boyutuna göre göreceli olarak değişir. Bu kök eleman da genelikle html etiketidir.

Root Parent (html) => font-size: 20px;

Parent (etkilemeyen) => font-size: 48px;

Child => font-size: 1rem; => 1em = 20px
