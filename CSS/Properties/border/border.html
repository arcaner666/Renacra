border

Uygulandığı elemanın etrafına istenilen özelliklerde bir çerçeve çizer.

Alt özelliklerle de ayrı ayrı çerçeve çizilebilir. Fakat bu özellik bunları tek satırda yapar.

Bu üç özelliğin kombinasyonuyla farklı şeyler elde edilebilir. border: <border-width> || <border-style> || <color>;

border: 3px solid red;

border-width çizginin kalınlığını belirler. px, em, rem, vh, vw ve bazı kelimeleri değer olarak alabilir.

border: thin solid red; => thin = 1px

border: medium solid red; => medium = 3px

border: thick solid red; => thick = 5px

border-style çizginin şeklini belirler.

border: 10px none red; => none varsayılan değerdir. Hiçbir çizgi çizmez.

border: 10px solid red;

border: 10px hidden red; => Çizgiyi çizer, kalınlık oluşur fakat çizgi görünmez.

border: 10px dashed red;

border: 10px dotted red;

border: 10px double red;

border: 10px groove red;

border: 10px ridge red;

border: 10px inset red;

border: 10px outset red;

color çizginin rengini belirler.

border: 10px solid red;

border: 10px solid blue;

border: 10px solid green;
