height

İçeriğin yüksekliğini belirler.

Değer olarak sadece uzunluk birimleri kabul eder.

height: 80px;

height: 2em;

height: 8ch;

height: 4cm;

height: 5vh;

height: 4rem;

height: 100%;

Eğer kapsayıcı bloğun yüksekliği ayarlanmamışsa ve öğe mutlak olarak konumlandırılmamışsa yüksekliğin değeri otomatik olarak hesaplanır.(Eğer blok içinde bir içerik varsa yükseklik içeriğin yüksekliğine eşit olur eğer yoksa sıfır olur.)

Child => height: 200px;

Parent'a yükseklik atanmadığı için parent'ın yüksekliği child'ın yüksekliğine eşit olur.

Eğer bloğun içerik kısmı ayarlanan yükseklikten daha fazla alana ihtiyaç duyuyorsa içeriğin davranışını overflow özelliği belirler.

Parent => height: 200px;

Child => height: 200px;

Parent ve child'ın yüksekliği eşit.

Parent => height: 200px;

Child => height: 250px;

overflow ayarlanmadığı için child'ın yüksekliği alta doğru 50px taştı.

Parent => height: 200px; overflow: hidden;

Child => height: 250px;

Parent => height: 200px; overflow: scroll;

Child => height: 250px;

Parent => height: 200px; overflow: auto;

Child => height: 250px;

Eğer bir child'ın yüksekliğine auto değeri girilirse yükseklik, özellikle belirtilmediği sürece child'ın içeriğinin yüksekliğine eşit olacaktır.

Parent => height: 150px;

Child => height: auto;

Eğer parent'ın yüksekliğine ölçülebilir bir yükseklik, child'ın yüksekliğine ise yüzde bir değer verilirse child'ın yüksekliği parent'ınkine göre oranlanır.

Parent => height: 150px;

Child => height: 95%;
